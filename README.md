# QMLHEP GSoC 2026 – Quantum Machine Learning Tasks

This repository contains my implementations for the evaluation tasks required by the Quantum Machine Learning for High Energy Physics (QMLHEP) program for Google Summer of Code 2026.

The tasks explore topics including:

- Quantum circuits
- Quantum machine learning
- Graph neural networks
- Hybrid quantum–classical models
- Applications to high energy physics datasets

## Repository Structure

Task1_Quantum_Circuits  
Task2_Graph_Neural_Networks  
Task3_Open_Commentary  
Task4_QGAN  
Task5_QGNN  
Task6_Quantum_Representation_Learning  

## Tools & Frameworks

- Cirq
- Pennylane
- TensorFlow Quantum
- PyTorch
- PyTorch Geometric

## Goal

The goal of this repository is to keep track of my exploration of how quantum computing and quantum machine learning can be applied to particle physics analysis.
